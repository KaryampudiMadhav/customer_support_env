## Troubleshooting

| Issue | Solution |
|-------|----------|
| Docker build fails | Check `python --version` (need 3.10+), verify all imports in `requirements.txt` |
| `openenv validate` fails | Run `openenv --version`, check openenv.yaml format, verify model imports |
| Grader returns invalid score | Ensure score is in [0.0, 1.0], handle edge cases (division by zero, missing fields) |
| Inference script times out | Check LLM API (rate limits), reduce number of test cases, use lower temperature |
| HF Space fails to start | Check Dockerfile PORT matches openenv.yaml, verify no hardcoded paths, test locally first |
